## What is RyunixJS?

RyunixJS is a modern UI framework designed to be **completely standalone**. While it draws inspiration from popular libraries like React and Preact, it implements its own custom reconciliation engine and provides a familiar API (Hooks, Components) with unique features for priority scheduling and server-side rendering.

Whether you're building a simple static site or a complex Single Page Application (SPA) with Server-Side Rendering (SSR), RyunixJS gives you the tools to do it efficiently.

## Framework Architecture

RyunixJS is built on several core principles:

- **Custom Reconciliation Engine**: A fiber-based reconciler that manages the virtual DOM and efficiently updates the real DOM
- **Priority-Based Scheduling**: Built-in support for concurrent rendering with priority levels (HIGH, NORMAL, LOW)
- **Comprehensive Hooks**: Specialized hooks like `usePersistentStore`, `useSwitch`, `useDebounce`, and `useThrottle`
- **Webpack Integration**: Powerful build system with custom plugins for routing, SSR, and SSG

## Overview

`@unsetsoft/cra` (Create Ryunix App) is the official scaffolder for RyunixJS applications. It provides a guided interactive experience to create modern, high-performance web applications with optimized default configurations.

## Quick Start

### Interactive Mode

Start the interactive setup with a guided experience:

```bash
npx @unsetsoft/cra@latest
```

### Specify Project Name

Create a project with a specific name:

```bash
npx @unsetsoft/cra@latest my-ryunix-app
```

<Note>
  Current version: **1.1.20-canary.4**
</Note>

## Command Line Flags

For more control over the setup process, use these flags:

### Version & Help

```bash
# Show current version
npx @unsetsoft/cra@latest --version

# Display help message
npx @unsetsoft/cra@latest --help
```

### Release Channel

```bash
# Use canary releases (experimental)
npx @unsetsoft/cra@latest my-app --canary

# Use stable releases (default)
npx @unsetsoft/cra@latest my-app --latest
```

### Feature Flags

```bash
# Initialize with Tailwind CSS
npx @unsetsoft/cra@latest my-app --tailwind

# Initialize with ESLint
npx @unsetsoft/cra@latest my-app --eslint

# Add VS Code settings
npx @unsetsoft/cra@latest my-app --vscode

# Combine multiple flags
npx @unsetsoft/cra@latest my-app --tailwind --eslint --vscode
```

### Available Flags

| Flag | Description |
| :--- | :--- |
| `-v, --version` | Output the current version of the CLI |
| `-h, --help` | Display the help message |
| `--canary` | Use the Canary channel for Ryunix dependencies |
| `--latest` | Use the Latest channel (stable, default) |
| `--tailwind` | Initialize with Tailwind CSS configuration |
| `--eslint` | Initialize with ESLint configuration |
| `--vscode` | Add VS Code settings for Ryunix extension |

## Project Structure

Every project scaffolded with `cra` includes:

```
my-ryunix-app/
├── app/                    # Application routes and pages
├── src/                    # Source code
│   ├── components/        # Reusable components
│   └── utils/            # Utility functions
├── public/                # Static assets
├── .ryunix/              # Build output (generated)
├── ryunix.config.js      # Ryunix configuration
├── package.json          # Dependencies and scripts
└── README.md            # Project documentation
```

## Configuration Files

The scaffolded project includes:

### ryunix.config.js

Pre-configured Ryunix settings:

```javascript
module.exports = {
  ssr: true,
  rootDir: 'src',
  buildDir: '.ryunix',
  port: 3000
};
```

### package.json Scripts

```json
{
  "scripts": {
    "dev": "ryunix dev",
    "build": "ryunix build",
    "start": "ryunix start"
  }
}
```

## Troubleshooting

### Permission Errors

If you encounter permission errors on Linux/macOS:

```bash
sudo npm install -g @unsetsoft/cra
```

### Clear npm Cache

If installation fails, try clearing the cache:

```bash
npm cache clean --force
npx @unsetsoft/cra@latest my-app
```

## Overview

`@unsetsoft/ryunix-devtools` is a browser extension that provides powerful debugging capabilities for RyunixJS applications. It integrates directly into your browser's DevTools to give you deep insights into your application's component tree, state, and performance.

<CardGroup cols={2}>
  <Card title="Component Tree" icon="diagram-project">
    Real-time visualization of your component hierarchy
  </Card>
  <Card title="Props Inspection" icon="magnifying-glass">
    Inspect and monitor component props and state
  </Card>
  <Card title="Hook Counter" icon="list-ol">
    Track hook usage and execution order
  </Card>
  <Card title="Auto Detection" icon="wifi">
    Automatically detects Ryunix applications
  </Card>
</CardGroup>

<Note>
  Current version: **1.3.0**
</Note>

## Installation

### Chrome / Edge

<Steps>
  <Step title="Download Extension">
    Clone or download the extension from the repository:
    ```bash
    git clone https://github.com/UnSetSoft/Ryunixjs.git
    cd Ryunixjs/packages/ryunix-devtools
    ```
  </Step>
  
  <Step title="Open Extensions Page">
    Navigate to:
    - Chrome: `chrome://extensions/`
    - Edge: `edge://extensions/`
  </Step>
  
  <Step title="Enable Developer Mode">
    Toggle "Developer mode" in the top right corner
  </Step>
  
  <Step title="Load Extension">
    Click "Load unpacked" and select the `packages/ryunix-devtools` folder
  </Step>
</Steps>

## Usage

### Opening DevTools

<Steps>
  <Step title="Open Browser DevTools">
    Press `F12` or right-click and select "Inspect"
  </Step>
  
  <Step title="Find Ryunix Tab">
    Look for the "Ryunix" tab in the DevTools toolbar
  </Step>
  
  <Step title="Auto Detection">
    The extension will automatically detect if the page is running a Ryunix application
  </Step>
</Steps>

<Note>
  If you don't see the Ryunix tab, make sure your application is built with Ryunix 1.3.0 or later.
</Note>

## Development

To build the extension from source:

```bash
cd packages/ryunix-devtools
npm run build
```

This creates a `devtools.zip` file ready for distribution.

## Troubleshooting

### DevTools Tab Not Appearing

<AccordionGroup>
  <Accordion title="Verify Ryunix Version">
    Make sure your application uses `@unsetsoft/ryunixjs` version 1.3.0 or higher.
    
    ```bash
    npm list @unsetsoft/ryunixjs
    ```
  </Accordion>
  
  <Accordion title="Check Development Mode">
    The DevTools hook is only active in development builds. Verify you're running:
    
    ```bash
    npm run dev
    ```
    
    Not the production build.
  </Accordion>
  
  <Accordion title="Reload Extension">
    Try reloading the extension:
    1. Go to `chrome://extensions/`
    2. Find Ryunix DevTools
    3. Click the reload icon
  </Accordion>
  
  <Accordion title="Clear Browser Cache">
    Clear your browser cache and reload the page:
    - Press `Ctrl+Shift+Delete` (Windows/Linux)
    - Press `Cmd+Shift+Delete` (Mac)
  </Accordion>
</AccordionGroup>

### Extension Not Detecting App

<Steps>
  <Step title="Check Console">
    Open the browser console and look for Ryunix initialization messages
  </Step>
  
  <Step title="Verify Installation">
    Ensure `@unsetsoft/ryunixjs` is installed:
    ```bash
    npm install @unsetsoft/ryunixjs
    ```
  </Step>
  
  <Step title="Reload Page">
    Hard refresh the page with `Ctrl+Shift+R` (Windows/Linux) or `Cmd+Shift+R` (Mac)
  </Step>
</Steps>

## Keyboard Shortcuts

While focused on the Ryunix DevTools panel:

| Shortcut | Action |
| :--- | :--- |
| `Ctrl+F` / `Cmd+F` | Search components |
| `↑` / `↓` | Navigate component tree |
| `→` | Expand component |
| `←` | Collapse component |
| `Ctrl+R` / `Cmd+R` | Refresh tree |

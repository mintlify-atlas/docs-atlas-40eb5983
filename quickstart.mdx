This guide will walk you through creating your first RyunixJS application, from scaffolding a new project to building interactive components.

## Create Your First App

<Steps>
  <Step title="Scaffold a new project">
    Use the official CLI to create a new RyunixJS application:

    <CodeGroup>
    ```bash npm
    npx @unsetsoft/cra@latest my-ryunix-app
    ```

    ```bash yarn
    npx @unsetsoft/cra@latest my-ryunix-app
    ```

    ```bash pnpm
    npx @unsetsoft/cra@latest my-ryunix-app
    ```
    </CodeGroup>

    The CLI will ask you a few questions:
    - **Project name**: Confirm or change the project name
    - **Package manager**: Choose npm, yarn, or pnpm
    - **Tailwind CSS**: Add Tailwind CSS for styling (optional)
    - **ESLint**: Add ESLint configuration (recommended)
    - **VS Code settings**: Add recommended VS Code settings (optional)
  </Step>

  <Step title="Navigate to your project">
    ```bash
    cd my-ryunix-app
    ```
  </Step>

  <Step title="Start the development server">
    <CodeGroup>
    ```bash npm
    npm run dev
    ```

    ```bash yarn
    yarn dev
    ```

    ```bash pnpm
    pnpm dev
    ```
    </CodeGroup>

    Your app will be available at `http://localhost:3000`
  </Step>
</Steps>

## Understanding the Project Structure

Your new RyunixJS project has the following structure:

```
my-ryunix-app/
├── app/
│   ├── index.ryx          # Main page component
│   ├── layout.ryx         # Root layout with metadata
│   └── errors.ryx         # Error boundary fallback
├── assets/
│   └── logo.svg          # Logo and static assets
├── public/               # Static files served as-is
├── styles/
│   └── global.css        # Global styles
├── ryunix.config.js      # RyunixJS configuration
├── package.json          # Project dependencies
└── .gitignore           # Git ignore rules
```

<Note>
Files with `.ryx` extension are JSX components that use RyunixJS. They work just like `.jsx` files but signal that they're RyunixJS components.
</Note>

## Your First Component

Let's examine the default `app/index.ryx` component that was created:

```jsx app/index.ryx
import { useStore } from '@unsetsoft/ryunixjs'
import logo from '../assets/logo.svg'

export default function Index() {
  const [count, setCount] = useStore(0)

  return (
    <main className="main">
      <img src={logo} alt="Ryunix Logo" className="logo" />
      <h1>Welcome to Ryunix</h1>
      <p className="subtitle">
        A fast, modern, and flexible JavaScript framework for building user interfaces.
      </p>
    </main>
  )
}
```

This component demonstrates:
- **Importing hooks** from `@unsetsoft/ryunixjs`
- **State management** with `useStore`
- **JSX syntax** for rendering UI
- **Asset imports** for images

## Add Interactivity

Let's make the component interactive by adding a counter:

<Steps>
  <Step title="Update the component">
    Modify `app/index.ryx` to add a button:

    ```jsx app/index.ryx
    import { useStore } from '@unsetsoft/ryunixjs'
    import logo from '../assets/logo.svg'

    export default function Index() {
      const [count, setCount] = useStore(0)

      return (
        <main className="main">
          <img src={logo} alt="Ryunix Logo" className="logo" />
          <h1>Welcome to Ryunix</h1>
          
          <div className="counter">
            <p>You clicked {count} times</p>
            <button onClick={() => setCount(count + 1)}>
              Click me
            </button>
          </div>
        </main>
      )
    }
    ```
  </Step>

  <Step title="Add some styles">
    Add CSS to `styles/global.css`:

    ```css styles/global.css
    .counter {
      margin: 2rem 0;
      text-align: center;
    }

    .counter button {
      padding: 0.75rem 1.5rem;
      font-size: 1rem;
      background: #3B82F6;
      color: white;
      border: none;
      border-radius: 0.5rem;
      cursor: pointer;
      transition: background 0.2s;
    }

    .counter button:hover {
      background: #2563EB;
    }
    ```
  </Step>

  <Step title="See the changes">
    Save the files and the browser will automatically reload with your changes thanks to Hot Module Replacement (HMR).
  </Step>
</Steps>

## Using Effects

Let's add a side effect to update the document title:

```jsx app/index.ryx
import { useStore, useEffect } from '@unsetsoft/ryunixjs'
import logo from '../assets/logo.svg'

export default function Index() {
  const [count, setCount] = useStore(0)

  useEffect(() => {
    document.title = `You clicked ${count} times`
  }, [count])

  return (
    <main className="main">
      <img src={logo} alt="Ryunix Logo" className="logo" />
      <h1>Welcome to Ryunix</h1>
      
      <div className="counter">
        <p>You clicked {count} times</p>
        <button onClick={() => setCount(count + 1)}>
          Click me
        </button>
      </div>
    </main>
  )
}
```

The `useEffect` hook runs whenever `count` changes, updating the browser tab title.

## Create a Reusable Component

Let's extract the counter into its own component:

<Steps>
  <Step title="Create a new component file">
    Create `app/components/Counter.ryx`:

    ```jsx app/components/Counter.ryx
    import { useStore } from '@unsetsoft/ryunixjs'

    export default function Counter({ initialCount = 0 }) {
      const [count, setCount] = useStore(initialCount)

      return (
        <div className="counter">
          <p>Count: {count}</p>
          <button onClick={() => setCount(count + 1)}>
            Increment
          </button>
          <button onClick={() => setCount(count - 1)}>
            Decrement
          </button>
          <button onClick={() => setCount(0)}>
            Reset
          </button>
        </div>
      )
    }
    ```
  </Step>

  <Step title="Use the component">
    Import and use it in `app/index.ryx`:

    ```jsx app/index.ryx
    import logo from '../assets/logo.svg'
    import Counter from './components/Counter.ryx'

    export default function Index() {
      return (
        <main className="main">
          <img src={logo} alt="Ryunix Logo" className="logo" />
          <h1>Welcome to Ryunix</h1>
          
          <Counter initialCount={0} />
        </main>
      )
    }
    ```
  </Step>
</Steps>

## Configure Metadata

The `app/layout.ryx` file controls the HTML document structure and metadata:

```jsx app/layout.ryx
import '../styles/global.css'

export const Metatags = {
  title: {
    template: '%s | My Ryunix App',
    default: 'My Ryunix App'
  },
  description: 'My awesome RyunixJS application',
  viewport: 'width=device-width, initial-scale=1.0',
  charset: 'UTF-8'
}

export default function RootLayout({ children }) {
  return children
}
```

The `Metatags` export controls:
- **title**: Page title with template support
- **description**: Meta description for SEO
- **viewport**: Viewport configuration for responsive design
- **charset**: Character encoding
